# 🐍💧🔫 Snake Water Gun Game

## 📌 Project Overview
The **Snake Water Gun Game** is a fun Python-based mini project inspired by **Rock-Paper-Scissors**.

### 🚀 This project includes:
- 🖥️ Command Line Version  
- 🎮 GUI Version (Tkinter)  
- 🌐 Web Version (HTML, CSS, JS)  
- 🔊 Sound Effects (Pygame)  
- 💾 Score Saving System  

---

## 🎮 Game Rules

| Choice | Beats |
|--------|------|
| 🐍 Snake | 💧 Water |
| 💧 Water | 🔫 Gun |
| 🔫 Gun | 🐍 Snake |

## ⚙️ Installation

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/snake-water-gun.git
2️⃣ Navigate to the project
cd snake-water-gun
3️⃣ Install dependencies
pip install pygame
▶️ How to Run
🖥️ CLI Version
python cli/main.py
🎮 GUI Version
cd gui
python gui_game.py
🌐 Web Version

Open:

web/index.html

in your browser

🕹️ How to Play
Choose:
🐍 Snake
💧 Water
🔫 Gun
Computer randomly selects a choice
Result is displayed instantly
Score updates automatically
🔊 Sound Effects
🔘 Button click → button_click.wav
🎉 Win → victory.wav
😞 Lose → defeat.wav
⚖️ Tie → draw.wav

👉 Sounds are stored in the assets/ folder and loaded using pygame.

💾 Score System
Scores are tracked during gameplay
Saved in:
data/score.txt
Previous scores can be viewed anytime

You Win 🎉
